# Fight Service

## Config

| ENV          | Description                      | Default |
| ------------ | -------------------------------- | ------- |
| SERVER_PORT  | Port of the login server         | 3000    |
| CONTEXT_PATH | Context path of the login server | /       |
| DB_FILENAME  | Path of DB filename              | db.json |

## Generate documentation

```shell
yarn doc
```

## Install dependencies

```shell
yarn install
```

## Run

```shell
yarn start
```
